## 1.2 Percentages in Business

### Learning Objectives
- Calculate percentages and percentage changes
- Understand profit margins
- Work with percentage points

### Core Concepts

**Calculating Percentages**
```
Formula: Percentage = (Part ÷ Whole) × 100

Example from the book:
Sales Revenue = $12,000,000
Profit = $600,000
Profit Percentage = (600,000 ÷ 12,000,000) × 100 = 5%
```

**Percentage Change**
```
Formula: % Change = ((New Value - Old Value) ÷ Old Value) × 100

Example:
Sales grew from $8,043,750 to $9,000,000
% Change = ((9,000,000 - 8,043,750) ÷ 8,043,750) × 100
        = (956,250 ÷ 8,043,750) × 100
        = 11.89%
```

**Gross Margin Calculation**
```
Gross Margin = (Gross Profit ÷ Revenue) × 100

From the book:
Revenue = $8,043,750
Cost of Goods Sold = $6,032,813
Gross Profit = $2,010,937
Gross Margin = (2,010,937 ÷ 8,043,750) × 100 = 25%
```

## 1.3 Working with Financial Ratios

### Learning Objectives

### Core Concepts

**Current Ratio**
```
Formula: Current Ratio = Current Assets ÷ Current Liabilities

From the book:
Current Assets = $1,740,000
Current Liabilities = $960,000
Current Ratio = 1,740,000 ÷ 960,000 = 1.81
```

**Quick Ratio (Acid-Test)**
```
Formula: Quick Ratio = (Current Assets - Inventory) ÷ Current Liabilities

Current Assets = $1,740,000
Inventory = $500,000
Current Liabilities = $960,000
Quick Ratio = (1,740,000 - 500,000) ÷ 960,000 = 1.29
```

**Debt-to-Equity Ratio**
```
Formula: Debt-to-Equity = Total Liabilities ÷ Total Equity

Total Liabilities = $2,360,000
Total Equity = $1,490,000
Debt-to-Equity = 2,360,000 ÷ 1,490,000 = 1.58
```

# Module 2: Algebra for Financial Analysis

## 2.1 Solving for Unknowns

### Learning Objectives
- Use algebraic equations to solve for unknowns
- Understand the relationship between variables in financial formulas
- Apply algebra to break-even analysis

### Core Concepts

**Break-Even Analysis**
The break-even point is where Total Revenue = Total Expenses

```
Let x = number of units sold
Revenue = Price × x
Expenses = Fixed Costs + (Variable Cost × x)

Break-even: Price × x = Fixed Costs + (Variable Cost × x)

Solve for x:
Price × x - Variable Cost × x = Fixed Costs
x(Price - Variable Cost) = Fixed Costs
x = Fixed Costs ÷ (Price - Variable Cost)
```

**Example from the book:**
"Break-even point is the level of sales required to produce operating results with no profits or losses."

If a company has:
- Fixed costs: $50,000 per month
- Gross margin: 40%
- Monthly break-even sales: $125,000

Check: $125,000 × 40% = $50,000

### Practice Problems

1. **Break-Even**
   - A company has fixed costs of $30,000 per month and a gross margin of 25%. What are the break-even sales?
   - If the company sells a product for $100 with variable costs of $60, and has fixed costs of $200,000, how many units must be sold to break even?

2. **Solving for Unknowns**
   - A company wants a 20% profit margin. If costs are $800,000, what should revenue be?
   - If gross margin is 30% and gross profit is $300,000, what is revenue?

3. **Real-World Application**
   The book discusses rapid growth: "Sizable increases in receivables and inventories can put a big dent in current cash flow."
   - If sales increase 25%, and receivables increase proportionally, what is the new receivables level if they were $750,000?

## 2.2 Linear Equations in Business

### Learning Objectives
- Model business relationships with linear equations
- Understand slope and intercept in financial contexts
- Use linear equations for forecasting

### Core Concepts

**Linear Relationship: Revenue as Function of Units Sold**
```
Revenue = Price × Units Sold
R = P × U

Example:
Price per unit = $50
Units sold = 1,000
Revenue = $50 × 1,000 = $50,000
```

**Cost Function**
```
Total Cost = Fixed Cost + (Variable Cost × Units)
C = FC + (VC × U)

Example:
Fixed Cost = $10,000
Variable Cost per unit = $20
Units = 1,000
Total Cost = $10,000 + ($20 × 1,000) = $30,000
```

**Profit Function**
```
Profit = Revenue - Cost
Profit = (P × U) - (FC + VC × U)
Profit = U(P - VC) - FC
```

### Practice Problems

1. **Revenue and Cost Functions**
   - A company sells a product for $75 per unit. Variable costs are $45 per unit, and fixed costs are $150,000. Write the profit function.
   - How many units must be sold to break even?

2. **Linear Modeling**
   - Sales grew from $7,312,500 to $8,043,750 over one year. What is the average monthly growth rate?
   - If sales continue at this rate, what will sales be next year?

3. **Real-World Example**
   From the book: "Cash flow from profit provides the seed capital for growth. It provides cash for replacing long-term operating assets as they wear out over time."
   - If an asset costs $100,000 and is depreciated over 5 years, what is the linear depreciation function?

# Module 4: Advanced Financial Analysis

## 4.1 Time Value of Money

### Learning Objectives
- Understand the concept of present and future value
- Calculate compound interest
- Apply time value to business decisions

### Core Concepts

**Future Value Formula**
```
FV = PV × (1 + r)^n

Where:
FV = Future Value
PV = Present Value
r = interest rate per period
n = number of periods
```

**Present Value Formula**
```
PV = FV ÷ (1 + r)^n
```

**Example from the book:**
If a company offers a discount of 2% for payment within 10 days instead of 30 days:
- Effective annual interest rate ≈ 36%

Check: 2% discount for 20 days early
Daily rate = 0.02/20 = 0.001
Annual rate = 0.001 × 365 = 0.365 = 36.5%

### Practice Problems

1. **Future Value**
   - A company invests $100,000 at 5% compounded annually for 3 years. What is the future value?
   - If you can earn 8% on an investment, how much will $50,000 grow to in 5 years?

2. **Present Value**

   - What is the present value of $200,000 received in 5 years at 10%?

3. **Real-World Application**

   - If you buy equipment for $100,000 and it generates $25,000 per year for 5 years, what is the net present value at 10%?

## 4.2 Growth and Decay Functions

### Learning Objectives
- Model exponential growth in business
- Understand compound growth rates
- Apply growth functions to revenue and expense projections

### Core Concepts

**Compound Growth Formula**
```
Future Value = Present Value × (1 + growth rate)^number of periods
```

**Growth Rate Calculation**
```
Growth Rate = (Future Value ÷ Present Value)^(1/n) - 1
```

**Example from the book:**
Sales grew from $8,043,750 to $9,000,000 in one year.
Growth Rate = (9,000,000 ÷ 8,043,750) - 1 = 0.1189 = 11.89%

### Practice Problems

## 4.3 Present Value of Cash Flows

### Learning Objectives
- Calculate present value of multiple cash flows
- Understand net present value (NPV)
- Apply NPV to business decisions

### Core Concepts

**Net Present Value (NPV)**
```
NPV = CF0 + CF1/(1+r) + CF2/(1+r)^2 + ... + CFn/(1+r)^n
```

**Discounted Cash Flow (DCF) Analysis**
```
PV = Σ CFt / (1+r)^t
```

**Example from the book:**
A business expects the following cash flows:
- Year 1: $200,000
- Year 2: $250,000
- Year 3: $300,000
- Discount rate: 10%
- Initial investment: $500,000

NPV = -500,000 + 200,000/1.1 + 250,000/1.1^2 + 300,000/1.1^3
    = -500,000 + 181,818 + 206,612 + 225,394
    = $113,824

### Practice Problems

1. **NPV Calculations**
   - Initial investment: $1,000,000
   - Cash flows: Year 1 $400,000, Year 2 $500,000, Year 3 $600,000
   - Discount rate: 12%
   Calculate NPV

2. **Comparing Projects**
   - Project A requires $100,000 and returns $40,000 per year for 3 years.
   - Project B requires $150,000 and returns $60,000 per year for 3 years.
   - Discount rate: 10%
   Which project has the higher NPV?

3. **Real-World Application**
   The book states: "Capital has a cost; interest is paid on debt capital, and the business needs to earn an adequate bottom-line profit (after interest) to justify the use of its owners' equity capital."
   - If a project costs $500,000 and is financed with 8% debt, what annual profit is needed to justify the investment?

# Module 5: Calculus Fundamentals

## 5.1 Introduction to Limits and Continuity

### Learning Objectives
- Understand the concept of limits
- Calculate limits of functions
- Understand continuity in business contexts

### Core Concepts

**Limit Definition**
```
The limit of f(x) as x approaches a is L if f(x) gets arbitrarily close to L as x approaches a.

lim f(x) = L
x→a
```

**Continuity**
A function is continuous if:
1. f(a) is defined
2. lim f(x) exists
3. lim f(x) = f(a)

**Business Application**
Marginal cost is the limit of average cost as production approaches a point.

### Practice Problems

1. **Limit Calculations**

3. **Real-World Application**

## 5.2 Derivatives: Rate of Change

### Learning Objectives
- Understand derivatives as rates of change
- Calculate basic derivatives
- Apply derivatives to marginal analysis

### Core Concepts

**Derivative Definition**
```
f'(x) = lim h→0 (f(x+h) - f(x))/h
```

**Power Rule**
```
If f(x) = x^n, then f'(x) = n × x^(n-1)
```

**Business Application**
- Marginal Revenue = derivative of Revenue function
- Marginal Cost = derivative of Cost function
- Marginal Profit = derivative of Profit function = 0 at maximum

### Practice Problems

1. **Basic Derivatives**
   - f(x) = x³, find f'(x)
   - f(x) = 5x² - 3x + 7, find f'(x)
   - f(x) = 2x⁴ - 6x², find f'(x)

2. **Business Derivatives**
   - Revenue = 100x - 0.5x². Find Marginal Revenue.
   - Cost = 1000 + 50x - 0.1x². Find Marginal Cost.
   - Profit = -0.5x² + 50x - 1000. Find Marginal Profit and find the maximum profit point.

3. **Real-World Application**
   The book discusses diminishing returns: "Sales have no theoretical limit; with smart marketing a business can keep increasing its sales."
   - If sales S(t) = 100t - 2t², find the rate of sales growth and when it stops growing.

## 5.3 Application: Optimization

### Learning Objectives
- Find maximum and minimum values of functions
- Apply optimization to business problems
- Understand economic implications of optimization

### Core Concepts

**Optimization Process**
1. Find critical points where f'(x) = 0
2. Determine whether each is maximum or minimum using f''(x)
3. Evaluate f(x) at critical points

**Second Derivative Test**
```
If f'(c) = 0 and f''(c) < 0, then f has a local maximum at c
If f'(c) = 0 and f''(c) > 0, then f has a local minimum at c
```

### Practice Problems

1. **Optimization**
   - Profit function: P(x) = -2x² + 120x - 500
     Find the maximum profit and the quantity that maximizes it.

   - Revenue function: R(x) = 50x - 0.5x²
     Find the revenue-maximizing quantity and maximum revenue.

2. **Business Optimization**
   - A company sells products at $200 per unit. The cost function is C(x) = 1000 + 80x - 0.1x².
     Find the profit-maximizing quantity and maximum profit.

   - Price depends on quantity: P = 500 - 2x. Cost = 1000 + 100x.
     Find the profit-maximizing quantity and price.

3. **Real-World Application**
   The book discusses inventory management: "Excessive inventory is expensive to maintain because it must be stored, insured, tracked, protected from theft."
   - If holding cost per unit is $5 per year and ordering cost is $100 per order, with annual demand of 10,000 units, find the optimal order quantity.

# Module 7: Multivariable Calculus

## 7.1 Partial Derivatives

### Learning Objectives
- Understand functions of multiple variables
- Calculate partial derivatives
- Apply to business problems with multiple inputs

### Core Concepts

**Partial Derivative**
```
∂f/∂x = derivative of f with respect to x, treating y as constant
∂f/∂y = derivative of f with respect to y, treating x as constant
```

**Business Application**
Production function: Q = f(K, L)
- ∂Q/∂K = marginal product of capital
- ∂Q/∂L = marginal product of labor

### Practice Problems

1. **Partial Derivatives**
   - f(x,y) = x²y + 3xy². Find ∂f/∂x and ∂f/∂y.
   - f(x,y) = 2x³y - 4xy². Find ∂f/∂x and ∂f/∂y.
   - f(x,y) = (x² + y²)^(1/2). Find ∂f/∂x and ∂f/∂y.

2. **Business Application**
   - Production: Q = 5K^(0.3)L^(0.7). Find ∂Q/∂K and ∂Q/∂L.
   - Cost: C = 100K + 50L. If K = 10 and L = 20, what is the marginal cost of capital?

3. **Real-World Application**
   The book discusses: "The assets show how the capital is deployed, and the other half of the balance sheet shows where the capital came from."
   - If profit function depends on debt (D) and equity (E): P = 0.1D + 0.15E - 0.001D² - 0.002E², find the profit-maximizing debt and equity.

## 7.2 Optimization with Multiple Variables

### Learning Objectives
- Find critical points of multivariable functions
- Use the second derivative test
- Apply to business optimization problems

### Core Concepts

**Critical Points**
```
∂f/∂x = 0 and ∂f/∂y = 0
```

**Second Derivative Test**
```
D = (∂²f/∂x²)(∂²f/∂y²) - (∂²f/∂x∂y)²
If D > 0 and ∂²f/∂x² > 0: minimum
If D > 0 and ∂²f/∂x² < 0: maximum
If D < 0: saddle point
```

### Practice Problems

   - f(x,y) = 3x² - 2xy + y² - 6x + 4y. Find and classify critical points.

2. **Business Application**

## 7.3 Lagrange Multipliers

### Learning Objectives
- Optimize functions subject to constraints
- Understand Lagrange multiplier method
- Apply to constrained business problems

### Core Concepts

**Lagrange Method**
```
Maximize f(x,y) subject to g(x,y) = 0

∇f = λ∇g
```
or
```
∂f/∂x = λ∂g/∂x
∂f/∂y = λ∂g/∂y
g(x,y) = 0
```

**Interpretation of λ**
λ is the marginal value of the constraint (shadow price).

### Practice Problems
